about the website:

this website is built with pure html and css and is hosted on netlify. it doesn't use any javascript.

the domain is from open domains and i really like their service :) (here is my pull request!)

i first started the website on my computer to test out nginx, but then i moved to neocities when i realized i wanted to expand this website.

this continued for a few months, then i discovered straw.page so i moved to there and left my neocities site, using it only for pure html projects.
